JavaScript Tutorial

Introduction to JavaScript

JavaScript is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions. It is most well-known as the scripting language for Web pages, but it's also used in many non-browser environments.

Features of JavaScript

  • Interactivity to web pages
  • Supports event-driven, functional, and imperative programming styles
  • Rich interfaces like drag and drop and sliders
  • Asynchronous programming capabilities with Promises and async/await

JavaScript Syntax

A simple JavaScript statement:

      
        console.log('Hello, world!');

Variables

Variables can be declared using var, let, or const.

      
        let message = 'Hello';
        const pi = 3.14;

Data Types

JavaScript supports dynamic types. Common types include:

  • Number
  • String
  • Boolean
  • Array
  • Object
  • Function
  • Null
  • Undefined

JavaScript Data Types

1. Number

Represents both integer and floating-point numbers.

let integer = 100;
  let floatingPoint = 100.5;
  console.log(integer);  // Outputs: 100
  console.log(floatingPoint);  // Outputs: 100.5

2. String

Textual data that can be defined using single, double quotes or backticks.

let singleQuote = 'Hello';
  let doubleQuote = "World";
  let templateLiteral = `Hello World`;
  console.log(singleQuote + " " + doubleQuote);  // Outputs: Hello World
  console.log(templateLiteral);  // Outputs: Hello World

3. Boolean

Logical entity that can have two values: true and false.

let isTrue = true;
  let isFalse = false;
  console.log(isTrue);  // Outputs: true
  console.log(isFalse);  // Outputs: false

4. Array

Used to store multiple values in a single variable. Arrays are zero-indexed.

let fruits = ['Apple', 'Banana', 'Cherry'];
  console.log(fruits[0]);  // Outputs: Apple
  console.log(fruits.length);  // Outputs: 3

5. Object

Represents instances of key-value pair collections.

let person = {
    firstName: 'John',
    lastName: 'Doe',
    age: 30
  };
  console.log(person.firstName);  // Outputs: John

6. Function

Defines a block of code that can be executed when "called". Functions can also be defined as expressions.

function greet(name) {
    return 'Hello ' + name + '!';
  }
  console.log(greet('Alice'));  // Outputs: Hello Alice!

7. Null

Represents the intentional absence of any object value.

let empty = null;
  console.log(empty);  // Outputs: null

8. Undefined

Indicates that a variable has not been assigned a value, or not declared at all.

let something;
  console.log(something);  // Outputs: undefined

Functions

Functions are defined using the function keyword.

      
        function greet(name) {
          return 'Hello ' + name + '!';
        }

JavaScript Functions

Functions in JavaScript are blocks of code designed to perform a particular task. They are executed when something invokes or calls them.

Function Declaration

To declare a function, use the function keyword followed by a name, followed by parentheses ().


  function greet() {
    console.log('Hello World!');
  }

You can call the function by using its name followed by parentheses.


  greet(); // Calls the function and logs 'Hello World!'

Function Parameters and Arguments

Functions can take parameters, which are used as variables inside the function. When you call the function, you pass arguments, which are the values given to the parameters.


  function sayHello(name) {
    console.log('Hello ' + name);
  }
  sayHello('Alice'); // Logs 'Hello Alice'

Return Values

Functions can return values that can be used in the program. The return statement determines the value the function returns.


  function add(a, b) {
    return a + b;
  }
  let sum = add(5, 3);
  console.log(sum); // Outputs 8

Function Expressions

A function can also be defined using an expression. Function expressions can be stored in a variable and are often used in situations where functions are used as values.


  const square = function(number) {
    return number * number;
  };
  console.log(square(4)); // Outputs 16

Arrow Functions

ES6 introduced arrow functions, which provide a more concise syntax for writing function expressions. They use an arrow (=>) to define the function.


  const multiply = (x, y) => x * y;
  console.log(multiply(5, 2)); // Outputs 10

Immediately Invoked Function Expressions (IIFE)

Sometimes, you might need to run a function immediately after defining it. This is done using IIFE.


  (function() {
    console.log('This function runs right away');
  })();

Higher-Order Functions

Functions that operate on other functions, either by taking them as arguments or by returning them, are called higher-order functions.


  function repeat(n, action) {
    for (let i = 0; i < n; i++) {
      action(i);
    }
  }
  repeat(3, console.log); // Outputs 0, 1, 2

Using Functions as Callbacks

Functions can be passed to other functions as callbacks, which are executed at a later time or under certain conditions.


  function greetUser(user, callback) {
    const msg = 'Hello ' + user;
    callback(msg);
  }
  greetUser('Alice', message => console.log(message)); // Outputs 'Hello Alice'

Event Handling

JavaScript can react to events like mouse clicks or key presses.

      
        <button onclick="alert('Button clicked!')">Click me!</button>

Object-Oriented Programming

JavaScript supports OOP with object constructors and the class keyword.

      
        class Car {
          constructor(brand) {
            this.carname = brand;
          }
          present() {
            return 'I have a ' + this.carname;
          }
        }

DOM Manipulation

JavaScript can manipulate the HTML content and style using the Document Object Model (DOM).

      
        document.getElementById("demo").innerHTML = "Hello JavaScript!";

JavaScript DOM Manipulation

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. JavaScript can manipulate the DOM to dynamically change the content and appearance of a web page.

Accessing Elements

You can access HTML elements using various methods provided by the document object.

  • document.getElementById(id): Selects an element by its ID.
  • document.getElementsByTagName(name): Selects all elements that have the specified tag name.
  • document.getElementsByClassName(name): Selects all elements that have the specified class name.
  • document.querySelector(selector): Returns the first element that matches a specified CSS selector(s).
  • document.querySelectorAll(selector): Returns all elements that match a specified CSS selector(s).

Changing Content

The innerHTML property can be used to get or set the HTML content of an element.


  document.getElementById('demo').innerHTML = 'Hello, DOM!';

Changing Styles

The style property is used to get or set the style of an element.


  document.getElementById('demo').style.color = 'red';

Creating Elements

You can create new elements with document.createElement(tagName).


  const newDiv = document.createElement('div');
  newDiv.innerHTML = 'New Element';
  document.body.appendChild(newDiv);

Removing Elements

To remove an element, you can use parentNode.removeChild(element).


  const parent = document.getElementById('parent');
  const child = document.getElementById('child');
  parent.removeChild(child);

Event Handling

JavaScript can capture and respond to user events like clicks, form submissions, and keyboard actions.


  document.getElementById('myBtn').addEventListener('click', function() {
    alert('Button clicked!');
  });

Working with Classes

Classes can be added, removed, or toggled on an element.


  const element = document.getElementById('myElement');
  element.classList.add('active');
  element.classList.remove('inactive');
  element.classList.toggle('visible');

Attribute Manipulation

You can get or set attributes of elements using getAttribute() and setAttribute().


  const anchor = document.getElementById('myLink');
  const hrefValue = anchor.getAttribute('href');
  anchor.setAttribute('href', 'https://www.example.com');

        }
